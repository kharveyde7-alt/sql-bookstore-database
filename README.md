# SQL Bookstore Database

## Project Overview

This project demonstrates fundamental SQL concepts using a bookstore database.

The SQL queries solve common business reporting problems while showcasing relational database concepts and best practices.

---

## Skills Demonstrated

- SQL Queries
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- CROSS JOIN
- UNION
- Subqueries
- Aggregate Functions
- GROUP BY
- HAVING
- Business Reporting

---

---

## Sample Queries

This project includes queries that:

- Find customers who ordered multiple books
- Calculate total books purchased by each customer
- Identify books that have never been ordered
- Find books priced above the average price
- Calculate total sales for each order
- Display books and their authors
- Generate customer/book combinations
- Combine data using UNION
- Identify customers who purchased books by a specific author
